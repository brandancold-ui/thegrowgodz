# The Grow Godz

Static multi-page website for the Grow Godz premium cannabis culture brand.

## Local preview

Serve the repository root with any static web server, then open `index.html`.

## Images

Production pages use optimized WebP assets under `assets/images/webp`. Source-quality files are preserved under `assets/images/originals`; see `assets/images/README.md` for the asset workflow.

## Community submissions

The Community form currently prepares a structured email to `info@thegrowgodz.com`, so GitHub Pages does not expose credentials or depend on an unconfigured endpoint.

The simplest secure hosted-form upgrade is a verified Formspree project connected to the Grow Godz inbox. Configure its real form action only after the project is created and verified; never place a private API key in this repository.
